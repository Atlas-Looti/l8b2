# Examples

Learn by example! Explore practical examples to see how to build games with l8b.

## Getting Started Examples

### [Basic Game](/examples/basic-game)
A simple game demonstrating core concepts: game loop, input handling, and rendering. Perfect for beginners!

### [Input Demo](/examples/input)
Interactive demo showing keyboard and mouse input handling.

## Framework Examples

### [Scene Management](/examples/scenes)
Learn how to organize your game into scenes (menu, gameplay, game over) and navigate between them.

## What You'll Learn

Each example includes:
- ✅ Complete, working code
- ✅ Step-by-step explanations
- ✅ Key concepts highlighted
- ✅ Suggestions for modifications

## Try It Yourself

All examples are designed to be:
1. **Copy-paste ready** - Just copy the code and run it
2. **Easy to modify** - Change values and see what happens
3. **Well-commented** - Understand what each part does
