# l8b Docs

Selamat datang di dokumentasi **l8b**, game engine web yang dibangun sepenuhnya dengan LootiScript.

## Why l8b?

l8b mempermudah developer membuat game berbasis web tanpa harus merangkai toolchain sendiri:

- **Ship Fast** – Tulis gameplay dengan file `.loot`, refresh browser untuk melihat perubahan. Tidak ada review store atau setup bundler manual.
- **Retain Players** – Engine menyiapkan canvas responsif, input multi-perangkat, dan loop stabil sehingga pengalaman konsisten di desktop maupun mobile.
- **Transact Seamlessly** – Integrasi storage dan IO memungkinkan penyimpanan skor, state, atau integrasi layanan eksternal tanpa boilerplate panjang.
- **Build Social** – Karena berbasis web, game mudah dibagikan lewat tautan; kamu bisa memanfaatkan API LootiScript untuk menambahkan fitur komunitas sendiri.

Tujuannya sederhana: fokus pada mekanik game, bukan konfigurasi tool.

## Prasyarat singkat

Sedikit pengalaman JavaScript dan game loop membantu, tapi contoh di dokumentasi akan fokus pada cara kerja LootiScript itu sendiri.
