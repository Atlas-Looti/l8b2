# API Reference

L8B provides various built-in APIs that can be used to build interactive games. These APIs provide access to features such as rendering, audio, input, asset management, and more.

## What is an API?

API (Application Programming Interface) is a collection of functions and objects provided by L8B to facilitate game development. Each API has a specific function:

- **Screen API** - For drawing and rendering on the screen
- **Audio API** - For playing sounds and music
- **Input API** - For reading input from keyboard, mouse, touch, and gamepad
- **Assets API** - For loading external files such as images, JSON, and text
- **System API** - For system information and program flow control
- **Storage API** - For persistent data storage
- **Scene API** - For managing scenes and routing
- **Sprites API** - For creating and managing sprites
- **Maps API** - For creating and managing tile maps
- **Palette API** - For managing color palettes
- **Standard Library** - Utility functions for Math, String, List, and JSON
- **Farcaster APIs** - For integration with Farcaster Mini Apps (Player, Wallet, EVM, Actions, HTTP)

Each API has complete documentation on its respective page. Use the sidebar to navigate to the specific API documentation you need.
