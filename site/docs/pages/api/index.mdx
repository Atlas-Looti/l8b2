# API Reference

L8B menyediakan berbagai API built-in yang dapat digunakan untuk membangun game interaktif. API-API ini memberikan akses ke berbagai fitur seperti rendering, audio, input, asset management, dan banyak lagi.

## Apa itu API?

API (Application Programming Interface) adalah kumpulan fungsi dan objek yang disediakan oleh L8B untuk memudahkan pengembangan game. Setiap API memiliki fungsi spesifik:

- **Screen API** - Untuk menggambar dan rendering di layar
- **Audio API** - Untuk memutar suara dan musik
- **Input API** - Untuk membaca input dari keyboard, mouse, touch, dan gamepad
- **Assets API** - Untuk memuat file eksternal seperti gambar, JSON, dan teks
- **System API** - Untuk informasi sistem dan kontrol alur program
- **Storage API** - Untuk menyimpan data secara persisten
- **Scene API** - Untuk mengelola scene dan routing
- **Sprites API** - Untuk membuat dan mengelola sprite
- **Maps API** - Untuk membuat dan mengelola tile maps
- **Palette API** - Untuk mengelola palet warna
- **Standard Library** - Utility functions untuk Math, String, List, dan JSON
- **Farcaster APIs** - Untuk integrasi dengan Farcaster Mini Apps (Player, Wallet, EVM, Actions, HTTP)

Setiap API memiliki dokumentasi lengkap di halaman masing-masing. Gunakan sidebar untuk navigasi ke dokumentasi API spesifik yang Anda butuhkan.
