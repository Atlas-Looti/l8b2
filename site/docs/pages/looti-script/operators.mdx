# Operators

LootiScript provides a rich set of operators for manipulating values.

## Arithmetic Operators

Perform mathematical calculations.

| Operator | Description | Example | Result |
|----------|-------------|---------|--------|
| `+` | Addition | `10 + 5` | `15` |
| `-` | Subtraction | `10 - 5` | `5` |
| `*` | Multiplication | `10 * 5` | `50` |
| `/` | Division | `10 / 5` | `2` |
| `%` | Modulo (Remainder) | `10 % 3` | `1` |
| `^` | Exponentiation | `2 ^ 3` | `8` |

## Comparison Operators

Compare two values and return `1` (true) or `0` (false).

| Operator | Description | Example |
|----------|-------------|---------|
| `==` | Equal | `5 == 5` |
| `!=` | Not Equal | `5 != 3` |
| `<` | Less Than | `3 < 5` |
| `>` | Greater Than | `5 > 3` |
| `<=` | Less Than or Equal | `3 <= 5` |
| `>=` | Greater Than or Equal | `5 >= 5` |

## Logical Operators

Combine boolean values. Remember `0` is false, everything else is true.

| Operator | Description | Example |
|----------|-------------|---------|
| `and` | Logical AND | `1 and 0` (returns 0) |
| `or` | Logical OR | `1 or 0` (returns 1) |
| `not` | Logical NOT | `not 0` (returns 1) |

## Assignment Operators

Assign values to variables, optionally performing an operation.

| Operator | Description | Equivalent To |
|----------|-------------|---------------|
| `=` | Assignment | `x = 5` |
| `+=` | Add and Assign | `x = x + 5` |
| `-=` | Subtract and Assign | `x = x - 5` |
| `*=` | Multiply and Assign | `x = x * 5` |
| `/=` | Divide and Assign | `x = x / 5` |
| `%=` | Modulo and Assign | `x = x % 5` |

## Bitwise Operators

Perform operations on binary representations of numbers.

| Operator | Description | Example |
|----------|-------------|---------|
| `&` | Bitwise AND | `5 & 3` |
| `\|` | Bitwise OR | `5 \| 3` |
| `<<` | Left Shift | `5 << 1` |
| `>>` | Right Shift | `5 >> 1` |

## Special Operators

### Delete
The `delete` operator removes a property from an object.

```lua
local player = object
  name = "Hero",
  health = 100
end

delete player.health
// player now only has 'name'
```

### Concatenation
Use `..` to concatenate strings.

```lua
local first = "Hello"
local second = "World"
print(first .. " " .. second)  // "Hello World"
```
